# פרויקט: מערכת ניהול מועדוני כדורגל

## רקע הפרויקט

אתם המנהלים של מועדון כדורגל חדש! המשימה שלך היא לבנות מערכת ניהול שתעזור לך לנהל שחקנים, לעקוב אחר ביצועים ולארגן את הצוות.

## נושאים המכוסים בפרויקט זה

- פונקציות (פונקציות רגילות ופונקציות חץ)
- אובייקטים (אובייקטים מורכבים)
- שיטות מערך (מסנן, מפה, מצא, מיון, הקטנה)
- מודולים (יבוא/ייצוא ES6)

## Common Mistakes to Avoid

1. **Don't modify the players.js file** - Only read from it
2. **Don't forget to export** your functions
3. **Don't forget to import** functions before using them
4. **Don't use var** - Use const or let
5. **Don't mutate original array** when sorting (use spread operator)
6. **Handle edge cases** - What if array is empty? Player not found?
